Study Assistant

Overview

Study Assistant is a Flask-based web application that helps students summarize study notes quickly and efficiently.

Features

- Paste study notes into a text area
- Generate concise summaries
- Clean and responsive user interface
- Built using Python and Flask

Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

Future Improvements

- Key term extraction
- Quiz generation
- Flashcard generation
- AI-powered summaries
